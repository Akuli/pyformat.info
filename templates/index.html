PyFormat Using % and .format() for great good!

Python has had awesome string formatters for many years now but the documentation here was far too technical. With this site we try to show you the most common use-cases covered by the old and the new string formatting API with pratical examples.

If you want to contribute more examples, feel free to create a pull-request on Github!
